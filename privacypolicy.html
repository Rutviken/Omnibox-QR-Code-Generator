Privacy Policy for Omnibox QR Code Generator

Introduction

Omnibox QR Code Generator respects your privacy and is committed to protecting your personal data. This privacy policy will inform you about how we handle your data when you use our Chrome extension.

Data Collection

No Data Collection: Omnibox QR Code Generator does not collect, store, or transmit any personal data or browsing information. The extension operates entirely locally within your browser and does not send any data to external servers.

How We Use Data

Since Omnibox QR Code Generator does not collect any data, there are no data usage practices to disclose.

Data Sharing

No Data Sharing: Omnibox QR Code Generator does not share any data with third parties. Since we do not collect any data, there is no data to share.

Permissions

The extension requires the following permissions to function properly:

  • activeTab: This permission allows the extension to access the URL of the active tab when the user interacts with the extension. This is necessary to generate the QR code for the current URL.
  • tabs: This permission allows the extension to manage and read the URL of the current tab to ensure the correct URL is captured for QR code generation. No data is collected or stored using these permissions.

Security

Omnibox QR Code Generator is designed to operate securely within your browser. The QR codes are generated locally, ensuring that your data remains private and secure.

Changes to This Privacy Policy

We may update this privacy policy from time to time. Any changes will be posted on this page, and we will notify users through our Chrome Web Store listing if significant changes are made.
